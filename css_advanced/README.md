### ğŸ”§ **Installation**

1. **Clone the repository**
   ```bash
   git clone https://github.com/MarkMMutunga/alx_html_css.git
   cd alx_html_css
   ```

2. **Navigate to the project**
   ```bash
   cd css_basic
   ```

3. **Open in browser**
